## Design Guidelines

### Color Scheme
- **Primary:** Purple #9333ea
- **Secondary:** Light Purple #a855f7
- **Dark Background:** #0D1117
- **Light Background:** #F3F4F6
- **Text Primary:** #f0f0f0 (dark mode), #111827 (light mode)
- **Accent Green:** #22c55e
- **Accent Orange:** #f97316
- **Accent Red:** #ef4444

### Typography
- **Font Family:** Inter (Google Fonts)
- **Headings:** Bold, 600-800 weight
- **Body:** Regular, 400-500 weight
- **Sizes:** 
  - Hero Title: 48-64px
  - Section Titles: 32-40px
  - Feature Titles: 20-24px
  - Body Text: 16-18px

### Style
- **Modern & Clean:** Minimalist design with focus on content
- **Glassmorphism:** Use subtle blur effects and transparency
- **Gradients:** Purple to dark gradients for hero and sections
- **Shadows:** Soft shadows for depth
- **Rounded Corners:** 12-20px border radius
- **Animations:** Smooth transitions and hover effects
- **Responsive:** Mobile-first design, works on all screen sizes

### Layout
- **Max Width:** 1200px for content
- **Spacing:** Generous padding and margins
- **Grid:** Use CSS Grid or Flexbox
- **Sections:** Clear separation with alternating backgrounds
- **Navigation:** Sticky header with smooth scroll
