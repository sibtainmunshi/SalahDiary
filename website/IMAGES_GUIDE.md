## 🔧 Image Optimization

### Before Upload:
1. **Resize** to correct dimensions
2. **Compress** using TinyPNG (https://tinypng.com)
3. **Convert** to WebP for better performance (optional)
4. **Rename** to match required names

### Compression Tools:
- **TinyPNG:** https://tinypng.com (Best for PNG)
- **Squoosh:** https://squoosh.app (Google's tool)
- **ImageOptim:** https://imageoptim.com (Mac app)
- **RIOT:** https://riot-optimizer.com (Windows)

### Target File Sizes:
- Hero image: < 500KB
- Screenshots: < 800KB each
- Total images: < 5MB

## 📱 Quick Screenshot Guide

### Using Your Phone:
1. **Open Salah Diary app**
2. **Navigate to screen** you want to capture
3. **Take screenshot:**
   - Samsung: Power + Volume Down
   - Pixel: Power + Volume Down
   - OnePlus: Power + Volume Down
4. **Transfer to computer:**
   - USB cable
   - Google Photos
   - Email to yourself

### Using Android Studio Emulator:
1. **Open Android Studio**
2. **Run app** in emulator
3. **Click camera icon** in emulator toolbar
4. **Save screenshot** to computer
5. **Repeat** for each screen

## 🎨 Design Tips

### For Best Results:
1. **Use consistent theme** (all dark or all light)
2. **Show realistic data** (not empty screens)
3. **Include visual elements** (icons, colors)
4. **Ensure text is readable**
5. **Use high contrast**
6. **Show app in action** (countdown running, etc.)

### What to Avoid:
- ❌ Blurry images
- ❌ Low resolution
- ❌ Empty screens
- ❌ Personal information
- ❌ Inconsistent themes
- ❌ Watermarks
